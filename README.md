# Minería de Datos — TUIA

Proyecto académico de **Minería de Datos** correspondiente a la **Tecnicatura Universitaria en Inteligencia Artificial (TUIA)**.

Este repositorio documenta de forma estructurada el proceso completo de análisis, preparación y exploración de datos, junto con los resultados obtenidos, siguiendo buenas prácticas de reproducibilidad y documentación técnica.

---

## Descripción

El trabajo abarca todas las etapas del proceso de minería de datos aplicadas a un conjunto de datos académico, desde la comprensión inicial del dominio hasta la síntesis de resultados y conclusiones finales.

El objetivo principal es aplicar técnicas de análisis de datos con criterios metodológicos claros, resultados verificables y documentación comprensible para su evaluación académica.

---

## Objetivos

* Comprender el dominio del problema y las características del conjunto de datos.
* Preparar los datos para su análisis (limpieza, transformación y selección de atributos).
* Realizar análisis exploratorio y/o modelado según corresponda.
* Presentar resultados y conclusiones con respaldo metodológico.

---

## Alcance

El proyecto es de carácter estrictamente académico y tiene fines educativos.

No se realizan implementaciones productivas ni despliegues en entornos reales.

## Requisitos

* Python 3.10 o superior
* Entorno virtual (recomendado)
* Dependencias listadas en `requirements.txt` (si aplica)

---

## Instalación y configuración

1. Crear y activar un entorno virtual (opcional):

```bash
python -m venv .venv
source .venv/bin/activate  # Linux / Mac
.venv\\Scripts\\activate     # Windows
```

2. Instalar dependencias:

```bash
pip install -r requirements.txt
```

## Uso

Ejecutar los notebooks en el orden sugerido dentro de la carpeta `notebooks/` para reproducir el análisis completo.

## Metodología

Se sigue un enfoque tradicional de minería de datos, estructurado en las siguientes etapas:

1. Comprensión del problema
2. Comprensión de los datos
3. Preparación de los datos
4. Análisis exploratorio y/o modelado
5. Evaluación
6. Conclusiones
